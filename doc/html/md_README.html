About

This program takes the number of steps and the final table of the game as input from the user and displays the initial table if possible as output. Run the program and give input like this:

Input

4 5 10
....*
..*.*
.*..*
..*.*

Output

..*.*
...**
**.**
*....

Build

Enter the following commands to build and run the program (CMake version >= 3.16 required)

cmake -S . -B build
cmake --build build
build/gol

Test

Enter the following commands to test using the Google Test Framework

cd test
cmake -S . -B build
cmake --build build
cd build && ctest
